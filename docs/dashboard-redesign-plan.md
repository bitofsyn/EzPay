# Dashboard 리디자인 계획

디자인 레퍼런스: `EzPay 금융 플랫폼.html` (Downloads)  
대상 파일: `frontend/src/pages/Dashboard.tsx`

---

## 현재 vs 목표 레이아웃

### 현재 레이아웃
```
[사이드바] | [헤더]
           | [히어로 카드(총 자산)] | [이달의 통계]
           | [내 계좌 카드 2개]     | [월별 지출 바 차트]
           | [자산 추이 차트]       | [월간 예산]
           | [최근 거래 (풀 행)]
```

### 목표 레이아웃 (디자인 기준)
```
[사이드바] | [헤더]
           | [히어로 카드 (개편)]   | [자주 보내는 친구]
           |                       | [이달의 통계]
           | [자산 추이 차트]       | [월별 지출 바 차트]
           | [최근 거래 (풀 행)]
```

---

## 단계별 수정 계획

### Step 1 — 사이드바 메뉴 추가

**파일:** `Dashboard.tsx` (`menuItems` 배열)

현재 메뉴: 대시보드, 내 계좌, 송금하기, 거래 내역, 설정  
목표 메뉴: 대시보드, 내 계좌, 송금하기, **친구**, 거래 내역, **알림**, 설정

변경 내용:
- `Users` 아이콘 import 추가 (lucide-react)
- `친구` 항목 추가 → 클릭 시 `/send` 이동 (친구 찾기 = 송금 상대 검색)
- `알림` 항목 추가 → 클릭 시 `/settings/notification` 이동

---

### Step 2 — 히어로 카드 개편

**위치:** Section 1 좌측 컬럼

현재: 그라디언트 배경 + 계좌 카드 2개 나열  
목표: 

```
┌─────────────────────────────────────────┐
│ 총 보유 자산                             │
│ ₩6,704,000                             │
│ 계좌 2개 · 전월 대비 +12.5%            │
│                                         │
│ ┌─────────────────────────────────┐     │
│ │ 토스뱅크  [대표]   전체 계좌 →  │     │
│ │ 1002-****-8901                  │     │
│ │ ₩1,284,000                     │     │
│ └─────────────────────────────────┘     │
│                                         │
│  [↑이체]  [↓충전]  [≡내역]  [♡친구]   │
└─────────────────────────────────────────┘
```

변경 내용:
- `mainAccount` 변수 추가 — `isMain=true`인 계좌 또는 첫 번째 계좌
- 대표 계좌 1개만 표시 (기존: 2개 카드 나열)
- 계좌번호 마스킹 표시 (`formatAccountNumber` 활용)
- "전체 계좌 →" 버튼 → `/accounts` 이동
- 액션 버튼 4개 추가: 이체(`/send`), 충전(`/accounts`), 내역(`/transactions`), 친구(`/send`)
- 아이콘: `ArrowUp`, `ArrowDown`, `AlignJustify`, `Users` (lucide-react)

---

### Step 3 — "자주 보내는 친구" 섹션 추가

**위치:** Section 1 우측 상단 (새 카드)

목표:
```
┌─────────────────────────┐
│ 자주 보내는 친구  전체보기│
│                         │
│  [김]  [이]  [박]  [최] │
│  민수  지현  서준  유진  │
└─────────────────────────┘
```

변경 내용:
- `frequentFriends` useMemo 추가
  - 거래 내역에서 **출금 거래**만 필터링
  - `receiverAccount.accountId` 기준으로 빈도 집계
  - 상위 4명 추출
  - 표시 이름: `receiverAccount.bankName` (현재 API에서 사용자 이름 미제공)
- 아바타: 이름 첫 글자 + 원형 배경
- "전체보기" 버튼 → `/transactions` 이동

---

### Step 4 — 우측 컬럼 레이아웃 변경

**위치:** Section 1 우측 컬럼

현재: 이달의 통계만 있음  
목표: 자주 보내는 친구 + 이달의 통계 (위아래로 쌓기)

변경 내용:
- 우측 컬럼을 `flex flex-col gap-5`로 구성
- 상단: 자주 보내는 친구 카드 (Step 3)
- 하단: 이달의 통계 카드 (기존 유지)

---

### Step 5 — Section 2 레이아웃 재구성

현재 Section 2: [내 계좌] | [월별 지출]  
목표 Section 2: [자산 추이] | [월별 지출]

변경 내용:
- 기존 "내 계좌" 섹션 제거 (히어로 카드에 통합됨)
- "자산 추이" 차트를 Section 2 좌측으로 이동
- "월별 지출" 바 차트를 Section 2 우측으로 이동

---

### Step 6 — 제거 항목

다음 섹션은 디자인에 없으므로 제거:
- **내 계좌** 카드 2개 섹션 (히어로 카드에 통합)
- **월간 예산** 섹션 (progress bar) — 별도 페이지로 이동 가능

최근 거래 섹션은 유지.
