# EzPay Feature Scope

## Primary Features

- Sign in and user session management
- Connect financial accounts
- Sync transactions from provider
- View normalized transaction history
- Review monthly and category-based spending
- Receive auto-generated spending insights
- Ask AI to explain insight changes

## Secondary Features

- Account list and balances
- Historical statistics views
- Notification settings

## Features to Freeze

- Transfer execution
- Transfer limit management
- Transfer cancellation UX
- Recipient-focused flow
- Transfer-specific admin management

## Portfolio Positioning

EzPay should be presented as:

`A personal finance analytics product that connects real financial transaction data, normalizes it, and turns it into explainable spending insights.`

It should not be presented as:

- A simple transfer app
- A generic chatbot for finance
- A fake-account banking demo
